## Installation
1. **Clone the repository:**
    ```bash
    git clone https://github.com/Ama4538/PortfolioV3.git
    cd PortfolioV3
    ```
2. **Install dependencies:**
    ```bash
    npm i
    ```
3. **Build and Start the server:**
    ```bash
    npm build
    npm preview
    ```
4. **Access the application:**
Open your browser and go to localhost
